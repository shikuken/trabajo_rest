# Monolithic REST API — TypeScript + SQLite (sin compilación nativa)

Proyecto monolítico REST API en **TypeScript + SQLite**, usando el paquete `sqlite` (no nativo).

### Endpoints:
- `/api/v1/projects`
- `/api/v1/tasks`
- `/api/v1/people`

### Cómo ejecutar
1. `npm install`
2. `npm run dev` (modo desarrollo)
3. O `npm run build` && `npm start` (modo producción)

Archivo de base de datos: `data/database.sqlite`
